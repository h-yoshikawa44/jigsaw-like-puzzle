# Jigsaw Like Puzzle

## 環境
base
- Node.js：20.9.0
- TypeScript：5.2.2
- React：18.2.0

Other major libraries
- Material UI：
  - core：5.14.18
  - icon：5.14.18
- React Konva：18.2.10

## 開発環境構築
- .env を作成
  ※内容は別途管理

- ライブラリのインストール
```bash
npm install
```

- ローカルサーバ起動
```bash
npm run dev
```

- ブラウザにアクセス
```
localhost:4173
```
