# jigsaw-like-puzzle

## 環境
base
- Node.js：14.17.3
- TypeScript：4.5.5
- React：17.0.2

Other major libraries
- Material UI：
  - core：5.4.2
  - icon：5.4.2
- React Konva：17.0.2-5

## 開発環境構築
- .env を作成
  ※内容は別途管理

- ライブラリのインストール
```
$ yarn install
```

- ローカルサーバ起動
```bash
$ yarn start
```

- ブラウザにアクセス
```
localhost:3000
```
